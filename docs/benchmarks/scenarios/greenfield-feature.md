# Scenario: Greenfield Feature
## Problem statement
Implement a new user-visible feature in a codebase with no existing implementation.

## Workflow steps
1. Define requirement and acceptance tests.
2. Implement feature.
3. Verify behavior.
4. Collect trace and audit evidence.

## Metrics to capture
- time to first working implementation
- trace coverage
- audit completeness
- recovery quality (if interrupted)

## Comparison axes
- no governance
- Spec Kit
- OpenSpec
- BMAD
- direct agent
- specsmith governed
