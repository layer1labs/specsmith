# Scenario: Audit Reconstruction
## Problem statement
Reconstruct a complete audit trail for a previously completed change set.

## Workflow steps
1. Select historical change.
2. Gather artifacts and traces.
3. Rebuild timeline and decision chain.
4. Assess reconstruction completeness.

## Metrics to capture
- reconstruction time
- trace coverage completeness
- audit completeness score
- recovery quality for missing context
