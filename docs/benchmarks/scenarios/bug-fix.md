# Scenario: Bug Fix
## Problem statement
Resolve a production defect with known failing behavior and expected correction.

## Workflow steps
1. Capture bug intent and constraints.
2. Reproduce failure.
3. Implement fix.
4. Validate tests and governance evidence.

## Metrics to capture
- mean time to resolution
- requirement/test trace linkage
- audit completeness
- recovery quality after interruption

## Comparison axes
- no governance
- Spec Kit
- OpenSpec
- BMAD
- direct agent
- specsmith governed
