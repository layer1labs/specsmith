# Scenario: Compliance-Sensitive Change
## Problem statement
Implement a change that affects compliance-relevant behavior and requires evidence packaging.

## Workflow steps
1. Capture regulated constraints.
2. Implement change with explicit test mapping.
3. Verify and audit.
4. Produce evidence export artifacts.

## Metrics to capture
- cycle time
- trace coverage depth
- audit completeness
- recovery quality
