# Scenario: Multi-Agent Implementation
## Problem statement
Deliver a medium-size feature using multiple specialized agents (plan, code, test, review).

## Workflow steps
1. Decompose work across roles.
2. Execute concurrent tasks.
3. Reconcile outputs.
4. Verify and audit final state.

## Metrics to capture
- time to merge-ready result
- cross-agent trace continuity
- audit completeness
- recovery quality under partial failures

## Comparison axes
- no governance
- Spec Kit
- OpenSpec
- BMAD
- direct agent
- specsmith governed
