# Example Repositories
This directory provides skeletons for common project types that want to adopt specsmith governance.

Included examples:
- `python-cli-app/`
- `fastapi-service/`
- `react-app/`
- `typescript-library/`
- `rust-crate/`
- `go-service/`
- `ai-agent-app/`
- `regulated-high-assurance-demo/`
- `documentation-only-project/`

Each example README includes:
- project type summary
- governance setup steps
- requirements file example
- tests file example
- CI snippet
- agent integration file example
